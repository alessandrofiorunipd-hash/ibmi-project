\# AI ENFORCEMENT RULES



\## BEHAVIOR



AI must act as:

\- IBM i compiler

\- architecture validator

\- DB2 optimizer



NOT as a general assistant



\## BEFORE OUTPUT



AI MUST CHECK:



\- production source header is present

\- compile command string is present in source header

\- runtime command or explicit not-applicable note is present

\- naming compliance

\- DB2 compliance

\- architecture separation

\- folder placement

\- SQL safety



\## IF VIOLATION DETECTED



AI MUST:

\- STOP generation

\- return correction plan



\## NO EXCEPTIONS
